# Holophonix Utils

A Blender addon providing various utility functions for Holophonix projects.

## Installation

1. Download the latest release
2. In Blender, go to Edit > Preferences > Add-ons
3. Click 'Install...' and select the downloaded zip file
4. Enable the addon
